Remembra v0.7.0

Enterprise Features & Cloud Integration

✨ Enterprise Features

Webhook System

Event-driven integrations with HMAC-SHA256 signing, automatic retry delivery with exponential backoff. Events include memory.created, memory.updated, memory.deleted, and entity.created.

RBAC (Role-Based Access Control)

Three roles: admin, editor, viewer. 12 granular permissions across memories, entities, webhooks, and admin. Scoped API keys with role assignment.

Memory Conflict Detection

Automatically detect contradictions in stored memories. Configurable strategies: update, version, or flag for manual resolution.

Audit Logging

Complete audit trail of all operations with export to JSON or CSV format. Role-protected admin endpoints.

📦 Import/Export System

Import from:
  • ChatGPT conversation exports
  • Claude conversation exports
  • Plain text files
  • JSON, JSONL, CSV formats
Export to:
  • JSON (full fidelity)
  • JSONL (streaming-friendly)
  • CSV (spreadsheet-compatible)

☁️ Cloud & Revenue

Stripe Billing Integration

Subscription management, usage-based metering, customer portal integration, and webhook handlers for billing events.

Plan Limits

Configurable limits per plan (memories, API calls, storage) with automatic enforcement and graceful degradation.

Spaces (Multi-tenancy)

Isolated memory spaces per organization with space-level settings, quotas, and cross-space queries for admins.

🔌 Plugin System

Extensible plugin architecture for custom integrations. Build your own plugins to extend Remembra's capabilities.
