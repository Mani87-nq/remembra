Remembra v0.13.0

Dashboard v2.0 & Team Collaboration

✨ Added

🔐 Two-Factor Authentication

TOTP-based 2FA for enhanced account security.

  • Enable via Settings → Security
  • Works with Google Authenticator, Authy, 1Password
  • Backup codes for account recovery

👥 Team Collaboration

Shared memory spaces with role-based access.

  • Create teams with Viewer/Member/Admin roles
  • Invite members with role picker
  • Link projects to teams
  • Share memories across team members

🛠️ Admin Dashboard

Full user management for platform operators.

  • View all users with plan, memory count, API keys
  • Delete, deactivate, or reset user passwords
  • Change user plans (Free/Pro/Team/Enterprise)
  • Search and filter users

📊 Activity Log

Security audit trail for compliance.

  • Track account and API activity
  • Color-coded by event type
  • JSON export for audits

🕵️ Entity Browser

Visual exploration of extracted entities.

  • Browse people, organizations, places, concepts
  • Click to see related memories
  • Entity counts and distribution

⏰ Timeline Timezone Fix

Proper local time display.

  • Dates now show in user's local timezone
  • "Today" header for current day
  • Times in 12-hour format (e.g., 09:57 AM)

📦 npm Package

TypeScript SDK for JavaScript developers.

  • npm install remembra
  • Full TypeScript support with types
  • Browser and Node.js compatible
  • Async/await API

🔧 Fixed

🔒 Security Improvements

  • RBAC Enforcement — Viewer role properly restricted from store/delete
  • SSRF Protection — Webhooks block private IP ranges
  • Error Sanitization — No Python exceptions leaked to clients
  • Bcrypt Performance — SHA256 cache mitigates O(n) timing attacks

🚀 Get Started

# Python
pip install remembra==0.13.0

# npm
npm install remembra

# Docker
docker pull remembra/remembra:0.13.0
docker run -d -p 8787:8787 remembra/remembra:0.13.0
