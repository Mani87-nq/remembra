Remembra v0.13.1

Cold Archive & Adaptive Thresholds

Added

Cold Archive Tier

Decayed memories can move out of active recall while staying queryable and restorable.

  • archived_memories table with archive metadata
  • Restore archived memories back to active storage with re-indexing
  • Keyword search and archive statistics for older context

Adaptive Thresholds

Pruning can adjust based on session mode, warm-up behavior, result quality, and memory density.

  • Session modes: exploratory, operational, and balanced
  • Conservative warm-up threshold for new sessions
  • Cleanup job integration through adaptive threshold configuration
