Remembra Changelog
April 25, 2026

Remembra v0.13.2

Product surfaces, dashboard control plane, and release safety upgrades for the next public push.

Added

  • Product Surface Refresh — The landing page now tells the product story through live memory, graph, product dock, and API system sections.
  • Dashboard Control Plane — A new operator overview brings memory health, API posture, graph signals, and recommended next steps into one focused surface.
  • Release Notes Coverage — Changelog pages and previews now cover the surface refresh and dashboard upgrade.

Changed

  • Truth Scrub — Landing and docs copy was tightened to avoid stale feature claims and unsupported compliance language.
  • Billing Checkout — Frontend Paddle checkout now initializes from runtime billing config instead of hardcoded tracked tokens.
  • Security Examples — Docs now use environment variables and placeholder-safe examples for API keys.
  • Git Safety — Ignore rules now cover local agent files, secrets, credentials, key files, and deployment context documents.

Install

pip install remembra==0.13.2
docker pull remembra/remembra:v0.13.2
docker run -d -p 8787:8787 remembra/remembra:v0.13.2
npm install remembra@0.13.2
