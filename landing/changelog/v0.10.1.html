Remembra v0.10.1

Centralized Credentials & Bridge Lifecycle

✨ Added

🔑 Centralized Credentials

API keys stored once, used everywhere.

  • ~/.remembra/credentials with chmod 600
  • API key saved on first install, auto-loaded for future installs
  • Priority: CLI arg > env var > credentials file
  • No need to pass --api-key every time

⚡ Slim Recall Mode

90% smaller payloads for token-constrained agents.

  • recall_memories(query, slim=True) returns only synthesized context
  • Reduces recall response from ~2KB to ~200 bytes
  • Perfect for agents with small context windows

🌉 Bridge Lifecycle Management

Full control over the local bridge for sandboxed agents.

  • remembra-bridge --stop gracefully stops running bridge
  • remembra-bridge --status checks if bridge is running and healthy
  • Port-in-use detection with clear error messages
  • Health check after startup (fails fast if unhealthy)
  • PID file management for clean process lifecycle

📦 Installation

# Via pip
pip install remembra==0.10.1

# Via Docker
docker pull remembra/remembra:0.10.1

# Quick start
remembra-install --all
